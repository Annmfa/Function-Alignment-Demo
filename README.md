# PA Function Alignment Demo

This repo now contains a minimal Hugo site for the paper _Bridging Perceptual and Analytic Dynamics via Function Alignment_.

## Local development

Run:

```bash
hugo server
```

Then open the local URL printed by Hugo, usually `http://localhost:1313/Function-Alignment-Demo/`.

## Production build

Run:

```bash
hugo
```

The generated site will be written to `public/`.

## Project structure

- Page content: `content/_index.md`
- Hugo template: `layouts/index.html`
- Styling and assets: `static/`
